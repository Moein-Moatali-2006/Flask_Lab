# Flask Blog

A simple blog web application built with **Flask**. Users can create an account, log in securely, publish blog posts, edit their profile, and follow other users.

## Features

- User registration and authentication
- Secure login/logout
- Create, edit, and delete blog posts
- Follow and unfollow other users
- Update user profile information
- Responsive UI built with Bootstrap
- Flask-based backend with template rendering

## Tech Stack

- Python
- Flask
- SQLAlchemy
- Bootstrap
- HTML5
- CSS3
- Jinja2
- SQLite

## Installation

```bash
git clone https://github.com/Moein-Moatali-2006/Flask_Lab

python -m venv venv

# Windows
venv\Scripts\activate

# Linux / macOS
source venv/bin/activate

pip install -r requirements.txt
```

## Run the Application

```bash
flask --app run
```

or

```bash
python run.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
